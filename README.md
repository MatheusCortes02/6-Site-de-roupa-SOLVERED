👕 SOLVERED Store
A SOLVERED é uma interface de e-commerce moderna voltada para o setor de vestuário. O projeto foca em uma experiência de usuário (UX) fluida, com design responsivo e estética minimalista de alto contraste.

🚀 Tecnologias Utilizadas
Este projeto foi construído utilizando as seguintes tecnologias:

HTML5: Estruturação semântica do conteúdo.

CSS3: Estilização avançada com foco em responsividade e design moderno.

JavaScript (Vanilla): Implementação de funcionalidades dinâmicas e interatividade.

LocalStorage: Persistência de preferências de interface (como modo escuro).

✨ Funcionalidades
Design Responsivo: Adaptável a dispositivos móveis, tablets e desktops.

Modo Escuro (Dark Mode): Toggle de alternância de tema com persistência de escolha do usuário.

Catálogo de Produtos: Exibição organizada de itens com foco em acessibilidade visual.

Navegação Fluida: Menu interativo e layout estruturado conforme as heurísticas de Nielsen.

🎨 Design e UI/UX
O projeto segue uma paleta de cores de alto contraste, utilizando tons escuros e detalhes que facilitam a leitura e a navegação. A interface foi planejada para ser intuitiva, priorizando a visibilidade dos produtos e a facilidade de interação.
