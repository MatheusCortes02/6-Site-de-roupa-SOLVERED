# SOLVERED

Site de loja de roupas desenvolvido para prática de front-end, com foco em design moderno, organização de produtos e experiência do usuário.

---

## Sobre o projeto

O SOLVERED é um projeto de e-commerce fictício criado para simular uma loja virtual de roupas.
A aplicação apresenta produtos, layout responsivo e navegação simples, servindo como base para evolução futura com funcionalidades completas de um sistema real.

---

## Funcionalidades

* Exibição de produtos
* Layout moderno
* Navegação simples e intuitiva
* Estrutura organizada de páginas
* Responsividade básica

---

## Tecnologias

* HTML5
* CSS3
* JavaScript

---

## Como rodar o projeto

```bash
# clonar repositório
git clone https://github.com/MatheusCortes02/6-Site-de-roupa-SOLVERED.git

# entrar na pasta
cd 6-Site-de-roupa-SOLVERED

# abrir no navegador
index.html
```
